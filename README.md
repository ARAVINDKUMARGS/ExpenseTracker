# Expense Tracker

A **Python-based Expense Tracker** to manage your incomes and expenses with detailed reports and visualizations. Perfect for personal finance management.

---

## Features

- Add Income
- Add Expense
- View Summary of finances
- Category-wise Report (with charts)
- Monthly Report (with charts)
- Search transactions
- Export to CSV

---

## Installation

1. Make sure **Python 3.x** is installed.
2. Install required packages:

```bash
pip install matplotlib
````

3. Clone the repository:

```bash
git clone https://github.com/ARAVINDKUMARGS/ExpenseTracker.git
cd ExpenseTracker
```

---

## Usage

Run the program:

```bash
python expense_tracker.py
```

Follow the on-screen menu:

```
1. Add Income
2. Add Expense
3. View Summary
4. Category-wise Report
5. Monthly Report
6. Exit
```
